## СТИЛЬ — ЗАФИКСИРОВАН ✓

> Полная спецификация в `STYLE_GUIDE.md`

### Краткая сводка

**Цвета:**
- Фон: `#f5f5f7`
- Карточки: `#ffffff`
- Тёмный (кнопки): `#1a1a2e`
- Акцент (ЛАЙМ): `#c8f542`
- Текст: `#1a1a2e` (основной), `#7a7f8a` (вторичный)

**Шрифты:**
- Заголовки: Unbounded (600)
- Основной: Manrope
- Лейблы: Chakra Petch

**Геометрия:**
- Скругления: 16px (кнопки, карточки), 12px (внутри)
- Отступы: кратны 8px
- Между секциями: 150px (десктоп), 80px (мобайл)

## ПРАВИЛА РАБОТЫ

### При начале сессии
1. Прочитать `CLAUDE.md`
2. Прочитать `PROJECT_CONTEXT.md`
3. Проверить `WORK_LOG.md`

### При работе
1. Редактируем ТОЛЬКО `src/`
2. Используем стиль из `STYLE_GUIDE.md`
3. Один блок за раз
4. Спрашиваем, если неоднозначно

### После утверждения
1. `git add .`
2. `git commit -m "описание"`
3. `git push`
4. Обновить `WORK_LOG.md`
